КАК ЗАГРУЗИТЬ В GITHUB

1. Распакуйте архив.
2. Откройте репозиторий interactive-book.
3. Удалите старые файлы из корня репозитория либо замените их.
4. Загрузите в КОРЕНЬ репозитория:
   index.html
   .nojekyll
5. Нажмите Commit changes.
6. Подождите 1–3 минуты.
7. Откройте:
   https://iohimtatyana1976-max.github.io/interactive-book/
8. Обновите страницу сочетанием Ctrl + F5.

Важно: файл index.html должен находиться именно в корне, а не внутри дополнительной папки.
