СКАЗОЧНАЯ СЛОВЕНИЯ
Версия для семейного просмотра

1. Распакуйте архив.
2. Откройте файл index.html в браузере.
3. Сначала проигрывается уже созданное интерактивное вступление.
4. Кнопка «Открыть книгу» переводит к первой легенде.
5. Для перелистывания используйте стрелки на экране или клавиши ← и →.
6. Кнопка «На весь экран» включает полноэкранный режим.
