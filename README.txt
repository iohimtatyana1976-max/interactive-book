КАК ЗАГРУЗИТЬ В GITHUB:
1. Распакуйте архив.
2. В репозитории удалите старые файлы или загрузите новые с заменой.
3. Обязательно загрузите ПАПКИ css, js и assets.
4. Нажмите Commit changes.
5. Подождите 1–3 минуты.
6. Откройте сайт и нажмите Ctrl+F5.
