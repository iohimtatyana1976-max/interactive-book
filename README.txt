БОЛЬШОЙ МИФ СЛОВЕНИИ — SITE VERSION 1.0

ЧТО ЗАГРУЖАТЬ В GITHUB
Загрузите в корень репозитория содержимое этой папки:

index.html
styles.css
script.js
assets/great-serpent.jpg

ВАЖНО
Папка assets должна сохраниться как папка.
Не загружайте только отдельные файлы без assets.

КАК ПРОВЕРИТЬ НА КОМПЬЮТЕРЕ
Откройте index.html двойным кликом.

КАК ОБНОВИТЬ GITHUB PAGES
1. Откройте репозиторий interactive-book.
2. Удалите или замените старые index.html, styles.css и script.js.
3. Загрузите новые файлы и папку assets.
4. Нажмите Commit changes.
5. Подождите 1–3 минуты и обновите опубликованный сайт.
