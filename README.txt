Сказочная Словения — V10

Главные изменения:
- интро теперь идёт как единая режиссёрская последовательность;
- увеличены паузы и замедлен темп;
- добавлены туман, световые частицы и более мягкие переходы;
- «Тс-с-с…» появляется ненадолго и растворяется;
- кнопка «Войти в сказку» появляется только в финале;
- все изображения встроены в один index.html;

Публикация:
1. Замените index.html в репозитории interactive-book.
2. Нажмите Commit changes.
3. Подождите 1–3 минуты.
4. Обновите сайт через Ctrl+F5.
