# Сказочная Словения — технический каркас

Готовый статический проект для GitHub Pages.

## Что уже работает

- кинематографичный вступительный экран;
- автоматический переход к Великой Липе;
- интерактивные точки двенадцати созвездий;
- сцена книги;
- 12 шаблонов сказок;
- адаптация под мобильный экран;
- поддержка `prefers-reduced-motion`;
- отсутствие сборщика и тяжёлого фреймворка.

## Как проверить локально

Не открывайте `index.html` двойным кликом, поскольку браузер может блокировать загрузку JSON.

Запустите локальный сервер:

```bash
python -m http.server 8000
```

Затем откройте:

```text
http://localhost:8000
```

## Публикация на GitHub Pages

1. Скопируйте содержимое проекта в корень репозитория `interactive-book`.
2. Сделайте commit и push.
3. В GitHub откройте **Settings → Pages**.
4. В качестве Source выберите **Deploy from a branch**.
5. Выберите ветку `main` и папку `/root`.
6. Сохраните настройки.

## Где менять данные

- `data/constellations.json` — названия и координаты созвездий.
- `stories/story01.html` … `story12.html` — тексты сказок.
- `css/main.css` — общий внешний вид.
- `js/intro.js` — длительность и логика пролога.
- `js/tree.js` — интерактивная навигация по Липе.
